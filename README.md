### Example Usage

#### Python
```python
import requests

    "system_size": 10,
    "temperature": 35,
    "actual_power": 40,
    "consumption": 50
}

response = requests.post('http://localhost:5000/predict', json=data)

if result['success']:
    print(f"Expected: {result['results']['expected_production']} kWh")
    print(f"Dust Loss: {result['results']['dust_loss_sar']} SAR/day")
```
